# Freight Sans Pro (self-hosted)

Place your Freight Sans Pro font files here. The site expects:

- `FreightSansPro-Book.woff2` (weight 400)
- `FreightSansPro-Medium.woff2` (weight 500)
- `FreightSansPro-Semibold.woff2` (weight 600)
- `FreightSansPro-Bold.woff2` (weight 700)

If your files use different names, update the `@font-face` rules in `src/app.css` to match.
